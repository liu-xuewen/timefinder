## TimeFinder

### 简介

- 分词基于SeGo
- 对自然语言（中文）提取时间

##### 使用

```
go get github.com/liu-xuewen/timefinder
```

```
var msg string
var extract []time.Time

// 初始化timefinder
segmenter := timefinder.New()

msg = " 6月9日有一场show要去观看"
// 解析话语词汇
extract = segmenter.TimeExtract(msg)
fmt.Println(msg)
fmt.Println(extract[0].Format(timeFormat))
```
